# Hackathon Project 2024 - UVM

This project was created during a hackathon. The goal was to build an AI-related program that uses facial expression recognition to detect a user's mood and connect that mood to music recommendations.

## Project Overview
The program uses computer vision tools to analyze facial expressions, such as face, eye, and smile detection. Based on the detected expression, the system connects the result to a mood-based music recommendation idea.

## Tools and Technologies
- Python
- Jupyter Notebook
- OpenCV
- Haar Cascade classifiers
- Computer Vision

## My Role
I contributed to the project development, testing, and presentation during the hackathon. This project gave me hands-on experience with computer vision, facial expression analysis, and AI-based user interaction.
